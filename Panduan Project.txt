Tools dan Library Yang Digunakan : Javascript, ReactJs, Ethers, Visual Studio Code.

Cara menjalankan project melalui Github :
- Pertama, Pastikan pada chrome nya sudah terinstall Extensions MetaMask 
- Download Project melalui link yang sudah saya lampirkan di email, atau
- Clone project meggunakan syntax 
"git clone https://github.com/fajarpp08/test-nusafinance.git"
- lakukan npm install pada direktori project yang sudah di clone tersebut
- kemudian, ketikkan "npm start" untuk menjalankan project
- Project running.

Cara menjalankan project melalui Download Google Drive :
- Pertama, Pastikan pada chrome nya sudah terinstall Extensions MetaMask 
- Download Project melalui link yang sudah saya lampirkan di email, atau
- Extract file yang sudah didownload
- Buka folder pada terminal command prompt atau terminal VSCode
- lakukan npm install pada direktori project tersebut
- kemudian, ketikkan "npm start" untuk menjalankan project
- Project running.
